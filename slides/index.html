Breakpoint Makes It Easier!


Breakpoint is a Compass extension to expedite the writing of media queries so that they can be quickly and efficiently be added into the CSS.

It can also output fall-back classes to use with Internet Explorer, because media-queries to not work

How to Use


$breakpoint-menu: 500px;

@include breakpoint($breakpoint-menu) {
  margin-right: 0;
}

This will output:

@media (min-width: 500px) {
  margin-right: 0;
}

Singularity & The Grid

The 960 Grid System

Create a layout with 12 columns, be able to divide those columns into halves, thirds, fourths, sixths, etc.


Easy.


We can create a simple layout like this within Singularity.

Basic Variables

  • $grids: All of the grid information, we will start with a basic layout (12 columns).

  • $gutters: The gutter information for the grids we are using. Expressed as a fraction of a single column.

  • $output: How we want to output the grid, 960 relies on 'float'

$grids: 12;
$gutters: 1/4;
$output: 'float';

Creating a Layout

We output a column with: 

@include grid-span($span, $position);

Where $span is the number of columns, and $position is where in the grid to position the object


#container {
  max-width: 60em;
  clear: both;
}

#left {
  @include grid-span(6,1);
}

#right {
  @include grid-span(6,7);
}
